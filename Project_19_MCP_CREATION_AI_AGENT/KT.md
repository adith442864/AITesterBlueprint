MCP
Modal context protocol 
* It is just a connection between the two different applications, or your LLM and the third-party libraries. 
* LLM can connect with third-party libraries like Slack, Jira, and everything by using modal context protocol. 

MCP has three components, if you remember. 
* Tools ( anything connection slack, weather, jira connection)
* Resource (file or some resource)
* Prompt ( promot, summarize)
