# Project 11: LangFlow Fundamentals

## Quick-Start and Starter Agents for QA Engineers

This project contains the LangFlow basics layer of the course before full RAG.

It is the place to start when students need to understand:

- how LangFlow works
- how components connect on the canvas
- how to test flows in the playground
- how to build simple prompt/model/output pipelines
- how to build starter QA agents before moving into RAG

---

## What's Inside

The main chapter is:

| # | Topic | Description |
|---|---|---|
| 1 | [LangFlow Quick-Start](./01_LangFlow_QuickStart/) | LangFlow basics, Groq examples, QA starter agents |

Inside the Quick-Start chapter, you now have:

2. prompt-based QA assistant
3. `RICE-POT` test case generator
4. Jira ticket fetch -> test plan + test cases
5. Jira story score -> HTML report
